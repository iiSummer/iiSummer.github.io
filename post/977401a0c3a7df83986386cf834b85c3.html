环境准备

1、Git Git主要是为了安装Hexo运行环境和提交Hexo所生成的静态文件到GitHub下
2、Node.js 运行环境咯
3、GitHub账号 主要是用到了GitHub Pages作为服务器

检查一下本地密匙，没有的话配置一下 

$ cd ~/. ssh 检查本机的ssh密钥

然后设置一下用户信息 

$ git config --global user.name "username"//用户名
$ git config --global user.email  "username@mail.com"//邮箱

登录GitHub建立一个和用户名一样的username.github.io这种可访问的站Repository，Github Pages的Repository名字是特定的，每个账号下面只能建立一个。

在本地创建一个目录，打开git，执行安装Hexo命令 

$ npm install -g hexo

初始化hexo，命令： 

$ hexo init

生成静态页面 

$ hexo generate（hexo g也可以）

启动本地服务，访问http://localhost:4000进行查看 

$ hexo server

配置_config.yml 

deploy:
     type: git
     repository: https://github.com/username/username.github.io.git
     branch: master

然后执行命令 

$ npm install hexo-deployer-git --save

然后部署 

$ hexo deploy

后面每次部署只需要这样操作就可以啦 

$ hexo clean
$ hexo generate
$ hexo deploy

然后访问https://username.github.io/就可以查看到你的博客啦
ps:每次发布会有几分钟延迟

Hexo一些常用命令

hexo new"postName" #新建文章
hexo new page"pageName" #新建页面
hexo generate #生成静态页面至public目录
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本
